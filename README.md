# Start of My Java 🚀

I have started coding in Java to upgrade my backend skills! This repository tracks my foundational progress as I transition into backend engineering, moving from core logic implementations to object-oriented programming concepts.

---

## 📂 Repository Contents

This repository features several small console applications focused on essential Java concepts:

*   **`VehicleDemo.java`** – An introduction to Object-Oriented Programming (OOP) fields, methods, and classes.
*   **`rps.java`** – A console-based Rock, Paper, Scissors game handling user string inputs and logic control.
*   **`evenodd.java`** – Simple conditional logic checking number properties using the modulo operator.
*   **`calculator.java` & `calculator1.java`** – Arithmetic operations, conditional math routing, and handling numeric user inputs.
*   **`number.java` & `test.java`** – Practical experimentation with variables, loops, and terminal output.

---

## 🛠️ Concepts Learned & Upgraded

*   **Java Syntax & Structure:** Writing clean code within standard Java classes and static main methods.
*   **Input Handling:** Utilizing `java.util.Scanner` to create interactive terminal experiences.
*   **Control Flow:** Mastering `if-else` branching paths to manipulate app behavior dynamically.
*   **OOP Foundations:** Instantiating objects, understanding constructors, and mapping real-world behaviors to code.

---

## 🚀 How to Run Locally

If you want to clone this project and test these files on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Shweta-coder8/Start_of_My_Java
   ```
   2. Navigate into the directory and compile any file (e.g., `evenodd.java`):
   ```bash
   javac evenodd.java
   ```
3. Run the compiled application:
   ```bash
   java evenodd
   ```
